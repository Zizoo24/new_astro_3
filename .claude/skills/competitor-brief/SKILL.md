# Competitive Landscape Brief (February 2026)

Load this context when making strategy, content, or SEO decisions.

## The Three-Player Map

### Al Syed (translationindubai.com)
- **Strategy:** English volume play. 350+ pages, ALL English, ZERO Arabic.
- **Strength:** #1-2 for "legal translation Dubai" through sheer page count (160+ blog posts, 60+ language pages)
- **Weakness:** Zero Arabic content, no schema markup, no sitemap, no authority-specific content
- **Platform:** Custom PHP/Laravel (not WordPress)
- **Reviews:** ~32 (GoodFirms)
- **Domain:** .com only (no .ae)
- **Note:** alsyedlegaltranslation.com does NOT exist. The actual domain is translationindubai.com.

### Alsun (alsuntranslation.com + 2 other domains)
- **Strategy:** Bilingual multi-domain. Only serious Arabic competitor.
- **Strength:** 40-70 Arabic pages, 256+ Google reviews at 4.9 stars, dominates Arabic SERPs
- **Weakness:** No authority-specific content, content spread across 3 domains fragments authority
- **Platform:** WordPress (WPML)
- **Domains:** alsuntranslation.com + legaltranslation-dubai.ae + alsuntranslation-riyadh.com

### OnlineTranslation.ae (Us)
- **Strategy:** Authority-specific depth + Arabic scale
- **Strength:** ~82 Arabic pages (most of any single domain), 4,500+ words/page, comprehensive schema, #1-2 for "MOJ translation Dubai"
- **Weakness:** 144 backlinks (vs 6,310 Al Syed, 2,772 Alsun), low Google review count
- **Platform:** Astro on Vercel, .ae domain

## Strategic Rules

1. **NEVER compete for "legal translation Dubai"** — Al Syed owns this with 350+ pages. Unwinnable.
2. **English = own the journey.** Target authority-specific queries (GDRFA, MOHRE, DHA, DIFC, ADJD, ADGM) where ZERO translation companies compete.
3. **Arabic = volume play.** Al Syed has 0 Arabic pages. Alsun has 40-70 across 3 domains. We have ~82 on one .ae domain. Optimize and scale to 120+.
4. **Abu Dhabi first** in all Arabic content. أبوظبي before دبي.
5. **Never mention competitor names** in published content.
