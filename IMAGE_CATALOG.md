# Image Catalog

All site imagery is consolidated under `public/assets/images` for easy reuse. Each folder groups related assets so future pages can reference consistent sources without hunting for files.

## Folder overview
- `decorative/` – connectors and other accent artwork.
- `hero-*.jpg` – general hero photos used across the site.
- `icons/` – favicons, touch icons, and utility UI icons.
- `logo/` – brand marks (emblems, optimized headers/footers, sidebar variants, glow/transparent logo files).
- `logos/` – partner and authority seals (Dubai Courts, MOJ, MOFAIC, DLD, Dubai Police).
- `og-image.jpg` – Open Graph preview image.
- `onedrive/` – hero scenes, tiles, service imagery, stamped document photos, contact shots, and upload/quote steps.
- `patterns/` – line, dot, and world map pattern SVGs (including the former `/assets/patterns` set now under `patterns/additional/`).
