## The One Process

All content creation follows the same workflow:

```
1. PLAN      → Define purpose, keywords, audience
2. WRITE     → Use appropriate prompt + style guide
3. VERIFY    → Complete pre-publish checklist
4. DEPLOY    → Build, commit, push
```

## Critical Rules (Quick Reference)

### Never Use

- Best, top, #1, elite, leading, premier
- Market leader, state-of-the-art, revolutionary
- Competitor names (never mention competitors)
- Exclamation points (!)

### Always Include

- MOJ License #701
- WhatsApp: +971 50 862 0217
- 8+ internal links per page
- FAQ section with schema

### Arabic-Specific

- Use formal أنتم form
- Prioritize Abu Dhabi mentions
- Use Arabic-Indic numerals (٦٠، ١٥٠)
