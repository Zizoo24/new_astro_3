## PART 3: THE UNIFIED WORKFLOW

### Step 1: Plan

Before writing, answer these questions:

```
□ What type of content? (Service page, blog, location, resource)
□ English or Arabic?
□ Primary keyword?
□ Target audience? (PRO, expat, business, individual)
□ Does the English version exist? (If Arabic)
□ URL structure determined?
```

**If you need SEO keywords, ASK THE USER to provide them.**

### Step 2: Structure

Every service page follows this structure:

```
1. HERO SECTION
   - H1 with primary keyword
   - 2-3 sentence intro
   - 3-5 bullet benefits
   - WhatsApp CTA

2. COMPLIANCE CHECKLIST
   - Prerequisites
   - Required documents
   - Accordion format

3. PROCESS STEPS
   - 4 clear steps
   - What happens at each stage

4. AFTER-CARE
   - What to do next
   - Related services

5. PRICING TIERS
   - Standard / Express / Premium
   - NO exact prices

6. FAQ SECTION
   - 6-15 questions
   - Schema markup
```

### Step 3: Write/Translate

**For English content:**
- Apply English style rules (Part 4)
- Include 8+ internal links
- Mention UAE entities (GDRFA, MOFA, DLD, DHA, MOHRE)

**For Arabic content:**
- Apply Arabic style rules (Part 5)
- Use formal أنتم form
- Prioritize: Abu Dhabi → Dubai → UAE
- Use Arabic-Indic numerals in body (٦٠، ١٥٠)

### Step 4: Verify

Use the checklist in Part 6 before publishing.

## PART 4: ENGLISH STYLE RULES

### Readability

| Rule | Limit |
|------|-------|
| Sentence length | Max 25 words (target 15-20) |
| Paragraph length | Max 3-4 sentences |
| Lists | Use bullets for 3+ items |
| Subheadings | Every 2-3 paragraphs |

### SEO Limits

| Element | Limit |
|---------|-------|
| Title tag | 60 characters max |
| Meta description | 155 characters max |
| H1 | One per page, include primary keyword |

### Title Formula

`[Primary Keyword] — [Benefit/Location] | OnlineTranslation.ae`

**Good:** Legal Translation Dubai — MOJ-Certified, 60-Min Delivery
**Bad:** Best Legal Translation Services in Dubai!

### Meta Description Formula

`[What we do] + [Key benefit] + [Trust signal or CTA]`

**Good:** MOJ-certified legal translation in Dubai. 60-minute delivery for standard documents. WhatsApp +971 50 862 0217.
**Bad:** We are the best translation company in Dubai!

### Internal Linking

- Minimum 8 internal links per page
- Descriptive anchor text (not "click here")
- Link to related services within silo
- Include cross-silo links

## PART 5: ARABIC STYLE RULES

### Formality

| Rule | Correct | Wrong |
|------|---------|-------|
| Addressing | أنتم (plural formal) | أنت (singular informal) |
| Verb form | تواصلوا، راسلونا، احصلوا | تواصل، راسلنا، احصل |
| We form | نقدم، نوفر، نلتزم | - |

### Approved Patterns

**Opening sentences:**
```
نقدم خدمات ترجمة قانونية معتمدة في أبوظبي ودبي.
نوفر ترجمة احترافية للوثائق والعقود في الإمارات.
نلتزم بتقديم ترجمة دقيقة ومعتمدة من وزارة العدل.
```

**CTAs:**
```
تواصلوا معنا اليوم.
راسلونا عبر واتساب.
احصلوا على عرض سعر.
```

### Typography

| Element | Format |
|---------|--------|
| Body numbers | Arabic-Indic: ٦٠ دقيقة، ١٥٠ درهم |
| Phone numbers | Western, in LTR span: `<span dir="ltr">+971 50 862 0217</span>` |
| Comma | Arabic: ، |
| Question mark | Arabic: ؟ |
| Semicolon | Arabic: ؛ |

### Geographic Priority

1. **Abu Dhabi (أبوظبي)** — mention prominently
2. Dubai (دبي)
3. UAE (الإمارات)
